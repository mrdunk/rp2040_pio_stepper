# UART Build Info at Power-On — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Print git branch name (with dirty indicator), build timestamp, and build username to the USB serial port at firmware power-on.

**Architecture:** A CMake script (`gen_build_info.cmake`) runs on every `make` invocation via `add_custom_target`, queries git for the branch name and dirty state, reads `$ENV{USER}` for the username, and writes a generated header `build_info.h` to the build directory. `stepper_control.c` includes this header and prints the info with two `printf` calls after the existing "UART up." message.

**Tech Stack:** CMake `execute_process` / `add_custom_target`, C `__DATE__` / `__TIME__` compiler built-ins, pico-sdk `printf` (routed to USB CDC via `pico_enable_stdio_usb`).

---

### Task 1: Write gen_build_info.cmake

**Files:**
- Create: `src/rp2040/gen_build_info.cmake`

- [ ] **Step 1: Create the cmake script**

Create `src/rp2040/gen_build_info.cmake` with this exact content:

```cmake
# gen_build_info.cmake
# Runs at build time (via add_custom_target) to generate build_info.h.
# Called with:
#   -DCMAKE_SOURCE_DIR=<repo root>
#   -DOUTPUT_FILE=<path to build_info.h>

# --- Branch name ---
execute_process(
    COMMAND git rev-parse --abbrev-ref HEAD
    WORKING_DIRECTORY ${CMAKE_SOURCE_DIR}
    OUTPUT_VARIABLE GIT_BRANCH
    OUTPUT_STRIP_TRAILING_WHITESPACE
    ERROR_QUIET
    RESULT_VARIABLE GIT_RESULT
)

if(NOT GIT_RESULT EQUAL 0 OR GIT_BRANCH STREQUAL "HEAD" OR GIT_BRANCH STREQUAL "")
    set(GIT_BRANCH "unknown")
endif()

# --- Dirty state ---
execute_process(
    COMMAND git status --porcelain
    WORKING_DIRECTORY ${CMAKE_SOURCE_DIR}
    OUTPUT_VARIABLE GIT_STATUS
    OUTPUT_STRIP_TRAILING_WHITESPACE
    ERROR_QUIET
)

if(NOT GIT_STATUS STREQUAL "")
    set(GIT_BRANCH "${GIT_BRANCH} (dirty)")
endif()

# --- Username ---
set(BUILD_USERNAME "$ENV{USER}")
if(BUILD_USERNAME STREQUAL "")
    set(BUILD_USERNAME "unknown")
endif()

# --- Write header ---
file(WRITE ${OUTPUT_FILE}
"/* Auto-generated by gen_build_info.cmake — do not edit */
#ifndef BUILD_INFO__H
#define BUILD_INFO__H
#define BUILD_GIT_BRANCH \"${GIT_BRANCH}\"
#define BUILD_USERNAME   \"${BUILD_USERNAME}\"
#endif  // BUILD_INFO__H
")
```

### Task 2: Wire gen_build_info.cmake into CMakeLists.txt

**Files:**
- Modify: `src/rp2040/CMakeLists.txt`

- [ ] **Step 1: Add custom target, include path, and dependency**

Open `src/rp2040/CMakeLists.txt`. After the `pico_generate_pio_header(...)` line (line 18) and before `target_sources(...)`, insert:

```cmake
# Generate build_info.h on every build by querying git and $ENV{USER}.
# stepper_control.c always recompiles so __DATE__/__TIME__ stay fresh.
add_custom_target(
    build_info
    COMMAND ${CMAKE_COMMAND}
        -DCMAKE_SOURCE_DIR=${CMAKE_SOURCE_DIR}
        -DOUTPUT_FILE=${CMAKE_BINARY_DIR}/build_info.h
        -P ${CMAKE_CURRENT_SOURCE_DIR}/gen_build_info.cmake
    BYPRODUCTS ${CMAKE_BINARY_DIR}/build_info.h
)

target_include_directories(stepper_control PRIVATE ${CMAKE_BINARY_DIR})
add_dependencies(stepper_control build_info)
```
